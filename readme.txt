# 📊 Binance Futures Trading Bot – Streamlit Version

This is a **web-based interface** for placing Market and Limit orders on the **Binance USDT-M Futures Testnet** using the official `python-binance` library. Built using **Streamlit**.

---

## 🧰 Features

- Place **BUY/SELL** orders with Market or Limit type
- Built-in logging of orders and API responses
- Secure API credential handling with `.env` file
- User-friendly form-based UI via Streamlit

---

## ⚙️ Requirements

- Python 3.8+
- Binance Futures Testnet account & API keys
- Streamlit and python-binance libraries
